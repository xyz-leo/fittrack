## [0.1.0] - 2025-11-16

### Adicionado

-   Protótipo inicial do FitTrack
-   Sistema de registro de usuários
-   Sistema de login (simulado para demonstração)
-   Dashboard do usuário com dados mockados

-   Modelo de dados `Student`
-   Templates HTML com Django
-   Estilização CSS com tema claro e laranja
-   Documentação completa no README.md

### Tecnologias

-   Python 3.12+
-   Django 5.0+
-   HTML5/CSS3
-   SQLite
-   Fonte Poppins (Google Fonts)
